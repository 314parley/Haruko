Mitsuba
=======

Project Mitsuba is an opensource (AGPLv3 licensed) image board software written in PHP and using MySQLi.

Installation
------------

To install Mitsuba you'll have to:
* Run ./install.php

Planned features
----------------

* logging actions
* captcha support
* catalog feature (like 4chan)
